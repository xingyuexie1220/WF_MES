# WF MES 用户操作手册（产线用户）

> 版本：v1.0.1.2 · 读者：产线操作员、条码文员、QA · 客户端：WF MES 桌面程序

---

## 1. 登录与退出

### 1.1 登录

1. 双击桌面 **WF MES** 图标（或 `WF.MES.WPF.exe`）
2. 输入账号、密码，点击 **登录**
3. 若提示「已在其他电脑登录」，确认是否在本机继续（会踢掉对方会话）
4. 若提示须修改密码，按向导设置新密码（至少 6 位）

### 1.2 注意

- **不要在一台电脑上打开两个 WF MES**（系统会自动拒绝）
- 长时间无操作可能因会话超时退出，请重新登录
- 退出请点击界面 **退出登录**

---

## 2. 界面说明

登录后左侧为 **功能模块**，点击模块展开 **二级菜单**，右侧为功能页面。

您能看到哪些菜单，由管理员在「角色权限分配」中配置。

---

## 3. 条码业务总流程

推荐按以下顺序理解完整链路：

```
客户维护 → 料号条码规则 → 上传资料 → QA 审核 → 条码打印 → （必要时）补打 / 查明细
```

| 步骤 | 菜单 | 谁操作 |
|------|------|--------|
| 1 | 客户维护 | 条码维护员 |
| 2 | 料号条码规则 | 条码维护员 |
| 3 | 条码资料审核（上传） | 文员 |
| 4 | 条码资料审核（确认/驳回） | QA |
| 5 | 条码打印 | 打印工位 |
| 6 | 条码明细 / 条码补打 | 查询或补打 |

> **重要**：料号须 QA **已确认** 后才允许打印；规则修改后会重新进入「待上传资料」。

---

## 4. 客户维护

**路径**：条码管理 → 客户维护

1. 点击 **新增** 或选中客户后 **编辑**
2. 填写客户名称、启用状态
3. **保存**

---

## 5. 料号条码规则

**路径**：条码管理 → 料号条码规则

1. 选择客户、输入料号
2. 配置规则段（固定文字、日期、流水号等），系统显示条码长度与预览
3. **保存**

保存后该料号 QA 状态会重置，需重新上传资料并审核。

若规则已有生成历史，修改 ResetKey 相关段时系统可能提示确认（避免流水号逻辑混乱）。

---

## 6. 条码资料审核

**路径**：条码管理 → 条码资料审核

### 6.1 状态说明

| 状态 | 含义 |
|------|------|
| 待上传资料 | 需上传图纸 + 打印实物图 |
| 待 QA 审核 | 资料齐全，等待 QA |
| 已确认 | 允许打印 |
| 已驳回 | 需重新上传资料 |

### 6.2 上传资料（文员）

1. 筛选料号，选中一行
2. 点击 **上传资料**（需有「保存附件」权限）
3. 选择 **图纸** 与 **打印实物图**（须同时上传）
4. 保存后状态变为 **待 QA 审核**

### 6.3 QA 审核

1. 选中 **待 QA 审核** 的记录，点击 **审核**
2. 查看图片与规则信息
3. **确认通过** 或 **驳回**（驳回须填写原因）

---

## 7. 条码打印

**路径**：条码管理 → 条码打印

1. 选择 **已 QA 确认** 的料号规则
2. 选择打印日期、数量（有单次上限，见界面提示）
3. 点击 **生成**，预览流水号区间与样例条码
4. 确认后生成条码单
5. 在结果窗口选择 **打印机**，点击 **打印**
6. 打印成功后生成单标记为 **已打印**

### 7.1 标签模板

- 模板文件：`程序目录\Labels\{料号}.btw`
- 若提示模板不存在，联系 IT 或条码维护员放入模板
- 可点击 **打开模板目录** 检查文件是否存在

---

## 8. 条码明细

**路径**：条码管理 → 条码明细

- 按客户、料号、生成单号、日期、打印状态筛选
- 查看生成单列表与流水范围
- 选中后可 **导出 CSV**（含条码清单）

> 系统仅保留近 **30 天** 内的生成记录，更早数据由后台任务清理。

---

## 9. 条码补打

**路径**：条码管理 → 条码补打

1. 查询 **已打印** 的生成单
2. 选择打印机，**补打**
3. 每张生成单仅可补打 **一次**，补打后状态为 **已补打**

---

## 10. 常见提示

| 提示 | 原因 | 处理 |
|------|------|------|
| 待 QA 确认 | 料号未审核通过 | 完成资料上传与 QA 审核 |
| 未找到标签模板 | 缺少 `{料号}.btw` | 联系维护员放入 Labels 目录 |
| 条码重复，请重试 | 极少见并发冲突 | 重新生成或联系 IT |
| 登录会话已失效 | 超时或被踢 | 重新登录 |
| WF MES 已在运行 | 重复打开程序 | 使用已打开的窗口 |
| 该生成单不可补打 | 未打印或已补打过 | 查明细确认状态 |
